## Description
Le projet « IPL social » contient une fonction de validation d’Email en TS.
1) La fonction doit vérifier qu’une chaine de caractère est bien une adresse email valide :
a. Il doit contenir au moins un @
b. Il doit contenir au moins un point dans le nom de domaine (et pas le dernier caractère)
c. Il ne peut contenir aucun espace
d. Il doit avoir du texte avant et après le @
2) Utilisation du TDD pour réaliser le tout


## Exécution des tests
Pour lancer les tests, utilisez la commande suivante dans le terminal à la racine du projet :
npm test
